Installation Guide for Docking System

Creating a New User

  1. Create a new user: 
    sudo adduser username

    During this process, the system will ask you to set a password for the new user and optionally provide some additional information (e.g., full name). Enter the password, and you can skip the rest by pressing Enter.

  2. Copy the default home directory content (if needed): 
    sudo cp -r /etc/skel/. /home/username
sudo chown -R username:username /home/username

    If you want the new user to have the default settings from the /etc/skel directory, execute the above commands.

  3. Add the user to the sudo group: 
    sudo usermod -aG sudo username

    Use this if you want the new user to have administrative privileges.

  4. Verify the user works correctly: 
    su - username

    You can log in to the new user's account and verify it works properly.

  5. Remove sudo privileges after completing installations: 
    sudo deluser username sudo

    Check that the user no longer has sudo access:

    sudo whoami

    You should see a message indicating that the user does not have permissions.

Copying Program Files

  1. Copy the entire dock folder to the home directory of the new user.
  2. Grant the appropriate permissions from the admin account:
    • Navigate to the user's home directory: 
      cd /home/username
    • Grant access rights to the dock folder: 
      chmod -R 755 /home/username/dock
sudo chown -R username:username /home/username/dock

Installing the System on a Computer without AutoDock Vina, P2RANK, Open Babel, PyMol

  1. Navigate to the installation directory:
    • The installation subdirectory contains the install.sh script, which downloads and installs Miniconda and sets up the auto_dock environment with Python 3.11.
    • Navigate to this directory: 
      cd /home/username/dock/installation
  2. Grant execution rights to the script: 
    chmod +x install.sh
  3. Run the installation script: 
    bash install.sh
  4. The script will install all required libraries and programs and perform their appropriate configuration.

Installing Necessary Packages for a New User (If the System is Already Configured)

  1. Navigate to the installation directory: 
    cd /home/username/dock/installation
  2. Grant execution rights to the script: 
    chmod +x mini_install.sh
  3. Run the installation script: 
    bash mini_install.sh
  4. The script will install all required libraries and programs and perform their appropriate configuration.
