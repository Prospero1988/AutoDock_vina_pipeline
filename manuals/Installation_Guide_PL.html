Instrukcja instalacji systemu dokowania

Kopiowanie plików programu

  1. Skopiuj cały folder dock do katalogu domowego (home) nowego użytkownika.
  2. Nadaj odpowiednie uprawnienia z konta administratora:
    • Przejdź do katalogu domowego użytkownika: 
      cd /home/nazwa_uzytkownika
    • Nadaj prawa dostępu do katalogu dock: 
      chmod -R 755 ~/dock
sudo chown -R nazwa_uzytkownika:nazwa_uzytkownika /home/nazwa_uzytkownika/dock

Instalacja systemu na komputerze bez zainstalowanych AutoDock Vina, P2RANK, Open Babel, PyMol

  1. Przejdź do katalogu instalacyjnego:
    • W podkatalogu installation znajduje się skrypt install.sh, który pobiera i instaluje Minicondę oraz tworzy środowisko auto_dock z Pythonem 3.11.
    • Przejdź do tego katalogu: 
      cd /home/nazwa_uzytkownika/dock/installation
  2. Nadaj prawa uruchomienia skryptu: 
    chmod +x install.sh
  3. Uruchom skrypt instalacyjny: 
    bash install.sh
  4. Skrypt zainstaluje wszystkie wymagane biblioteki i programy oraz wykona ich odpowiednią konfigurację.

Instalacja niezbędnych pakietów dla nowego użytkownika (jeśli system jest już skonfigurowany)

  1. Przejdź do katalogu instalacyjnego: 
    cd /home/nazwa_uzytkownika/dock/installation
  2. Nadaj prawa uruchomienia skryptu: 
    chmod +x mini_install.sh
  3. Uruchom skrypt instalacyjny: 
    bash mini_install.sh
  4. Skrypt zainstaluje wszystkie wymagane biblioteki i programy oraz wykona ich odpowiednią konfigurację.
