Instrukcja instalacji systemu dokowania

Tworzenie nowego użytkownika

  1. Utwórz nowego użytkownika: 
    sudo adduser nazwa_uzytkownika

    W trakcie wykonywania tego polecenia system poprosi o podanie hasła dla nowego użytkownika oraz kilku opcjonalnych danych (np. pełne imię i nazwisko). Wpisz hasło, a resztę możesz pominąć, naciskając Enter.

  2. Skopiuj zawartość standardowego katalogu domowego (jeśli potrzeba): 
    sudo cp -r /etc/skel/. /home/nazwa_uzytkownika
sudo chown -R nazwa_uzytkownika:nazwa_uzytkownika /home/nazwa_uzytkownika

    Jeśli chcesz, aby nowy użytkownik miał kopię domyślnych ustawień z katalogu /etc/skel, wykonaj powyższe polecenia.

  3. Dodaj użytkownika do grupy sudo: 
    sudo usermod -aG sudo nazwa_uzytkownika

    Jeśli chcesz, aby nowy użytkownik miał uprawnienia administratora.

  4. Sprawdź, czy użytkownik działa poprawnie: 
    su - nazwa_uzytkownika

    Możesz zalogować się na konto nowego użytkownika i sprawdzić, czy działa poprawnie.

  5. Po zakończonych instalacjach odbierz uprawnienia sudo: 
    sudo deluser nazwa_uzytkownika sudo

    Sprawdź, czy użytkownik nie ma już dostępu do sudo:

    sudo whoami

    Powinien pojawić się komunikat o braku uprawnień.

Kopiowanie plików programu

  1. Skopiuj cały folder dock do katalogu domowego (home) nowego użytkownika.
  2. Nadaj odpowiednie uprawnienia z konta administratora:
    • Przejdź do katalogu domowego użytkownika: 
      cd /home/nazwa_uzytkownika
    • Nadaj prawa dostępu do katalogu dock: 
      chmod -R 755 /home/nazwa_uzytkownika/dock
sudo chown -R nazwa_uzytkownika:nazwa_uzytkownika /home/nazwa_uzytkownika/dock

Instalacja systemu na komputerze bez zainstalowanych AutoDock Vina, P2RANK, Open Babel, PyMol

  1. Przejdź do katalogu instalacyjnego:
    • W podkatalogu installation znajduje się skrypt install.sh, który pobiera i instaluje Minicondę oraz tworzy środowisko auto_dock z Pythonem 3.11.
    • Przejdź do tego katalogu: 
      cd /home/nazwa_uzytkownika/dock/installation
  2. Nadaj prawa uruchomienia skryptu: 
    chmod +x install.sh
  3. Uruchom skrypt instalacyjny: 
    bash install.sh
  4. Skrypt zainstaluje wszystkie wymagane biblioteki i programy oraz wykona ich odpowiednią konfigurację.

Instalacja niezbędnych pakietów dla nowego użytkownika (jeśli system jest już skonfigurowany)

  1. Przejdź do katalogu instalacyjnego: 
    cd /home/nazwa_uzytkownika/dock/installation
  2. Nadaj prawa uruchomienia skryptu: 
    chmod +x mini_install.sh
  3. Uruchom skrypt instalacyjny: 
    bash mini_install.sh
  4. Skrypt zainstaluje wszystkie wymagane biblioteki i programy oraz wykona ich odpowiednią konfigurację.
