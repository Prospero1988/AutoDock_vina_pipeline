Installation Guide for Open-PyMOL on Windows

1. Download and Install Miniconda

  1. Open your web browser and navigate to the Miniconda download page.
  2. Download the appropriate Miniconda installer for Windows (64-bit recommended).
  3. Once downloaded, run the installer and follow the on-screen instructions. It's recommended to install for "Just Me" and check the option to add Miniconda to your PATH environment variable during installation.
  4. After installation, open the Command Prompt to verify the installation by typing: 
    conda --version

2. Initialize Conda, Create and Activate the Open_PyMOL Environment

  1. Open the Command Prompt.
  2. Initialize Conda by running: 
    conda init
  3. Restart the Command Prompt to apply the changes.
  4. Create a new Conda environment named open_pymol by executing: 
    conda create -n open_pymol python=3.10
  5. Activate the newly created environment: 
    conda activate open_pymol

3. Download and Install Open-PyMOL via Conda

  1. With the open_pymol environment activated, install Open-PyMOL by running: 
    conda install -c conda-forge openpymol
  2. Wait for the installation to complete. Conda will handle all necessary dependencies.

Running Open-PyMOL

  1. Open the Command Prompt.
  2. Activate the open_pymol environment: 
    conda activate open_pymol
  3. Launch Open-PyMOL by typing: 
    pymol
  4. Open-PyMOL should now start, and you can begin using it for your molecular visualization needs.
